Evrmore Authentication System

A secure blockchain-based authentication system powered by Evrmore

Welcome to the Evrmore Authentication Demo

This demo showcases how to authenticate users with their Evrmore wallets using cryptographic signatures instead of passwords.

How it works:

  1. The server generates a unique challenge for your Evrmore address
  2. You sign the challenge with your wallet's private key
  3. The server verifies the signature and authenticates you
